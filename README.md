#EV-Global Company Introduction

EV-Global is a fictional Chinese electric vehicle maker with design, engineering and manufacturing headquarters in Shenzhen, China. Its ex-China mainland PRC global headquarters are based in Hong Kong, which handles strategy, technology and sales outside of China. In addition to Hong Kong which functions as global headquarters, there are US, Canada, EU and Malaysia offices.

Each office has its own directory, which includes company introduction, personnel list with titles, macro policies which affect the company, and records changes and events which happen in each office.

Because China is the global leader in EV technology and manufacturing, the US, Canada and EU have had poliicy changes which affect EV-Global, not just on the sales and marketing levels, but also on key technologies, especially battery technology. The Chinese government considers these to be sensitive national technologies, and requires permission to export for these technologies to be transferred outside China.

The company wants to expand its sales presence outside China, but it must comply with Chinese government policy, and follow local laws in each country where it has a presence.

##EV-Global Purpose on Github

This company serves as an exhibit for Microsoft 365, Entra(identity management) and Purview (ediscovery, compliance and data governance) use cases which are hosted in other Github repositories. Each office has its own diretory/folder which includes separate files covering the company introduction, changelog which records company-specific changes and events, macro-poliicy changes which affect the company, and personnel names and job titles. These use cases will reference scenario situations which happen at EV-Global, and will exhibit how these Microsoft tools are used in the processing of different situations.

Since Chinese data regulations require that all Chinese corporations use Chinese applications and store all data on Chinese servers in China, the Chinese company does not use Microsoft technologies except in Hong Kong, which has been the only exception in China for historical reasons from before 1997. This is why the Chinese company and its domestic data is not included in this repository, and only references Hong Kong and other non-China offices.
